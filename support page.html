Common Problems and solutions

Issue: PC will not turn on after build is made. Solution: Make sure PSU cables are properly set and then try flicking the power button on the box.

Issue: CPU running hot. Solution: Make sure CPU has enough thermal paste on or plastic film is removed.

Issue: Case running hot. Solution: Make sure fans are facing right way and pulling from the front and pushing out the back

Issue: Bad computer performance. Solution: Make sure RAM is set in channels 2 and 4 for optimal usage.
